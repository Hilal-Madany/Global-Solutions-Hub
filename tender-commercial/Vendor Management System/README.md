# 🏛️ Vendor Management System (F-VEN-2025-01)
**Professional Assessment & Registration Portal** *Compliant with Indian Railways | EPC | PSU | Government & Defence Standards*

## 📄 Overview
This project is a high-fidelity automation of the **F-VEN-2025-01 (Issue 3.0 / Rev 00)** Assessment Sheet. It provides a techno-commercial evaluation framework for core industries, ensuring that vendor registration is data-driven, transparent, and audit-ready.



## 🎯 Scoring Matrix (Part L)
The system evaluates vendors based on a **100-point Scoring Matrix** as defined in the official evaluation guidelines:

| Evaluation Section | Max Marks | Criteria |
| :--- | :--- | :--- |
| **A. Organisational Soundness** | 25 | Years of operation, manpower strength, and infrastructure. |
| **B. Financial Soundness** | 20 | 3-year audited turnover, net worth, and current ratios. |
| **C. Quality Management (QMS)** | 25 | ISO 9001/14001/45001 certifications and QC manuals. |
| **D. Approvals & References** | 15 | RDSO/PSU/EIL approvals and past performance certificates. |
| **E. After-Sales Service** | 10 | Service network and technical support availability. |
| **F. Delivery Commitment** | 5 | Track record of timely project completion. |

## 🌟 Evaluation Outcomes & Star Ratings
The system automatically generates a status based on the final score:
* **⭐ Approved (70 - 100%):** Immediate registration and issuance of Approval Letter.
* **🟡 Conditionally Approved (50 - 69%):** Provisional registration with a 12-month review cycle.
* **❌ Not Approved (< 50%):** Rejection; vendor may re-apply after 6 months with improvements.

## 🚀 Key Features
- **Multi-Part Data Entry:** Covers Part A (Identity) through Part I (Contractor Assessment).
- **Statutory Validation:** Integrated checks for GST, PAN, and MSME/Udyam details.
- **Perfect-Print PDF:** Generates a formal assessment report ready for signing and stamping.
- **Cloud Database:** Automatic synchronization with Google Sheets for procurement teams.
- **Confidentiality:** Built with "For Official Use Only" data protocols.
