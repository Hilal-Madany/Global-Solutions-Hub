# 🧭 Technical Career Guidance Roadmap

### Professional Growth Path
1. **Trainee (0-6M):** Learning Product Specs & Drawing Reading.
2. **Junior Engineer (1-2Y):** Site Surveys, Costing & Estimation.
3. **Senior Engineer (2-5Y):** Negotiation, Closure & Project Coordination.
4. **Manager/Head (5Y+):** Strategy, International Export & Team Leadership.

### Core Skill Matrix
* **Technical:** BOQ preparation, FIDIC basics, Site measurement.
* **Commercial:** Negotiation, Objection handling, Payment recovery.
